# 🏥 AL-NOOR Hospital Management System

A modern Hospital Management System built using **React.js**. This project provides a clean and responsive interface for hospital information, doctor profiles, and appointment booking.

## 🚀 Features

- 🏥 Responsive Hospital Website
- 👨‍⚕️ Doctor Information Section
- 📅 Appointment Booking Form
- ℹ️ About Hospital Section
- 📱 Mobile Responsive Design
- 🎨 Modern UI Design

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- CSS
- Vite

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/sultan092-ahmad/hospital-react.git
```

Go to project folder:

```bash
cd hospital-react
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Open your browser:

```
http://localhost:5173
```
